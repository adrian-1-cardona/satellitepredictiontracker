# Satellite Tracker: Enterprise-Grade Visualization & UI for NASA Presentation

## Executive Summary

This specification outlines the transformation of the Satellite Tracker frontend into a mission-critical, enterprise-grade application suitable for NASA presentation and potential operational deployment.

**Key Requirements**:

### Part 1: 3D Satellite Visualization (Senior Software Engineer)

1. **Camera Positioning**: Pull back to show full Earth (currently too zoomed)
   - Current: `[0, 0.2, 7.4]` (too close, claustrophobic)
   - Target: `[0, 0.3, 9.2]` (proper orbital perspective)

2. **Realistic Space Environment**
   - True black background (no blue tint)
   - HDRI space/galaxy skybox for authentic NASA-quality appearance
   - No decorative star patterns overlay

3. **Satellite Graphics**
   - High-quality glTF/GLB models (Sketchfab, NASA 3D Resources)
   - PBR materials with roughness/metalness maps
   - ~100 satellites across 3-5 orbital shells
   - Different scales for ISS (prominent), constellation satellites (visible but smaller)

4. **Animation Speed**
   - LEO satellites: 60-90 seconds per orbit (realistic but visible)
   - MEO satellites: 3-5 minutes per orbit
   - GEO satellites: Nearly stationary
   - ISS: 30-45 seconds rotation (realistic solar panel tracking)

### Part 2: Frontend UI & Design (Senior UI/UX Designer)

1. **Color System - Professional Semantic**
   - Primary: #00b4d8 (NASA blue) - use for CTAs, important actions
   - Primary-dark: #0096c7 (hover state)
   - Success: #10b981 (green for online/success states)
   - Warning: #f59e0b (amber for caution)
   - Danger: #ef4444 (red for errors/offline)
   - Remove: All neon/trendy colors (mint, lime)

2. **Button Design**
   - Flat, no gradients (except subtle background gradients max 2)
   - Primary buttons: Solid color, no glow effects
   - Clear semantic meaning (blue=action, green=success, red=danger)
   - Proper hover/focus states (not playful, professional)

3. **Background & Aesthetics**
   - Remove: Decorative star pattern overlay
   - Remove: Overly vibrant radial gradients
   - Keep: Very subtle background, true black space feeling
   - No "AI-generated vibes" (no excessive gradients, no trendy colors)

4. **Landing Page**
   - Professional hero content overlay on 3D scene
   - Hero title + subtitle + CTAs
   - Call-to-action buttons: "Get Started" (primary), "Sign In" (secondary)
   - Responsive design (mobile, tablet, desktop)

5. **Accessibility**
   - WCAG 2.1 AA minimum (AAA preferred)
   - Color contrast: 4.5:1+ for text (7:1+ for AAA)
   - Keyboard navigation fully functional
   - Focus indicators clearly visible
   - No mouse-only interactions

6. **Status Indicator**
   - Simplify styling (no backdrop blur, professional look)
   - Online: Green (#10b981)
   - Offline: Red (#ef4444)
   - Checking: Gray (--muted)
