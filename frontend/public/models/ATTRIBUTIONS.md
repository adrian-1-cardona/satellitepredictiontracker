# 3D Model Attributions

This directory holds 3D model assets used by the Satellite Tracker frontend.
Every binary asset shipped here must be listed with source and provenance.

NASA content is used in a factual, non-endorsement context. Per NASA media
guidance, NASA should be acknowledged as the source.
