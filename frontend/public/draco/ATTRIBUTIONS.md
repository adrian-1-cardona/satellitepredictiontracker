# Draco Decoder Attributions

These files are copied from `three/examples/jsm/libs/draco/gltf/` so the
frontend can decode NASA GLB assets that use Draco mesh compression.

## Assets

- `draco_decoder.js`
- `draco_decoder.wasm`
- `draco_wasm_wrapper.js`
